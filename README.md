# Practica1

## Descripción
Programa en C que calcula el valor de PI utilizando la serie de Leibniz.

## Fórmula
\[\pi = \frac{4}{1} - \frac{4}{3} + \frac{4}{5} - \frac{4}{7} + \frac{4}{9} - \frac{4}{11} + \cdots + \frac{4}{n}\]
